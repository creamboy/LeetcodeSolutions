191.	Reverse Bits
Reverse Integer

Reverse digits of an integer.

Example1: x = 123, return 321
Example2: x = -123, return -321

Use Math: Time ~ O(N), Space ~ O(1)
check overflow each step

public int reverse(int x) {
    int val = 0, num = x;
    while (num != 0) {
        if (Math.abs(val) > Integer.MAX_VALUE / 10)   return 0;
        val = val * 10 + num % 10;
        num = num / 10;
    }
    return val;
}
